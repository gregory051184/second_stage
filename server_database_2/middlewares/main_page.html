Роутинг для фильмов

  • /film без параметров выводит все фильмы

    используя этот url с параметрами можно фильтровать ответ сервера

    Параметры для запросов

    ?film_id= по уникальному ключу ?name= по названию ?eng_title= по английскому названию ?production_country = по стране создания ?year_of_production = по году создания ?slogan = по слогану ?fk_director = по режиссёру ?fk_script = по сценаристу ?fk_producer = по продюсеру ?fk_operator = по оператору ?fk_composer = по композитору ?fk_artist = по художнику ?fk_montage = по монтажёру ?budget = по бюджету ?marketing = по затратам на рекламу ?usa_fees = по сборам в США ?other_world_fees = по сборам в мире ?premiere_in_russia = по дате премьеры в России ?premiere_in_world = по дате премьеры в мире ?release_on_dvd = по дате выпуска на DVD ?fk_age_restrictions = по ограничениям по возрасту ?fk_mpaa_rating = по рейтингу MPAA ?duration = по продолжительности ?translations = по переводу ?fk_subtitles = по языку субтитров ?fk_video_quality = по качеству изображения

  • /film/genre без параметров выводит все фильмы, с параметром сортирует по жанрам

    Параметры для запросов

    ?title=Название жанра

  • /film/person без параметров выводит все фильмы, с параметром сортирует по личностям

    Параметры для запросов

    ?second_name=Фамилия лица

  • /film/actor без параметров выводит все фильмы, с параметром сортирует по актёрам

    Параметры для запросов

    ?second_name=Фамилия актёра

  • /film/dubbing_actorбез параметров выводит все фильмы, с параметром сортирует по актёрам дубляжа

    Параметры для запросов

    ?second_name=Фамилия актёра дубляжа

  • /film/country для получения фильмов стране просмотра

    Параметры для запросов

    ?title= по наазванию страны

  • /create/film для занесения в базу новых фильмов

    необходимо внести все данные по фильму, кроме film_id(он создаётся автоматически)

    ВАЖНО! Дополнительно вы можете внести несколько массивов для заполнения связанных таблиц (можно и не указывать):

    genres здесь вы перечисляете id жанров, к которым будет относиться новый фильм

    actors здесь вы перечисляете id актёров, которые будут относиться к новому фильму

    dubbing_actors здесь вы перечисляете id актёров дубляжа, которые будут относиться к новому фильму

    countries здесь вы перечисляете id стран и кол-во зрителей, причём внутри массива countries они будут выступать отдельными массивами

    { ...необходимые данне о фильме, "genres": [1, 2], "actors": [4, 5, 6, 7, 9, 12], "dubbing_actors": [2, 8, 11, 15], "countries": [[1, 201000], [2, 373000], [6, 542000]] }

  • /update/film для внесения изменений в фильмы

    необходимо передать в виде параметра film_id того фильма, который хотите изменить ?film_id=

  • /delete/film для удаления фильмов

    необходимо передать в виде параметра film_id того фильма, который хотите удалить ?film_id=
